Seam Resizing

Matlab + HTML/CSS

Demo

When we imagine resizing an image, we usually think of stretching/compressing or cropping an image. What we would like, however, is to resize an image without there being artifacts, maintaining as much of the focus of the original image as possible. This is done by seam insertion and removal.

Image Blending

Matlab + HTML/CSS

Demo

I played around with changing images by using gaussian or laplacian filters. I sharpened an image using the unsharp masking technique. I blended the high frequencies of one image with the low frequencies of another. I blended two images together by using a filter over a mask and then by stacking filters together.

Pinhole Camera

Shoebox + HTML/CSS

Demo

With a team of 2 others, I built a pinhole camera using a shoebox, some construction paper, and some tape. We put a camera into the box and took pictures of the inside of the box, which captured the light from the surroundings.

Colorize Prokudin-Gorskii Glass Plate Images

Matlab + HTML/CSS

Demo

I used Matlab to colorize Prokudin-Gorskii glass plate images by aligning each image layer (corresponding to either R, G, or b) using a sum-of-squared-distance metric. I then enhanced these images using some autocontrasting and cropped out borders.

ClapCam Application

Android

Download

Source Code

A clap activated camera application for Android devices. Using the TarsosDSP library, the app listens for two distinct claps. Upon hearing such an activity, a timer goes off which eventually triggers the camera. This allows the user to take pictures from afar, perfect for taking group pictures or selfies from far away.
