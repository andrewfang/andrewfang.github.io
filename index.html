Bop With Friends

Java: Android

Download

Packages

A project I worked on with "Team We're Feeling Lucky" at the 2013 Google Android Camp. It is a gesture and motion rich game that is optimized for Android tablet devices. This app won first place at the Android Camp competition.

Battleship

Java

Source Code

A personal project that I am currently working on. It is a text-based game of Battleship that can be played by 0-2 human players in the command line prompt. Remaining spots are filled in by artificial intelligence players who can setup and play the game based on a predetermined strategy.

DrawIt Whiteboard

Javascript/HTML/CSS

Demo

Source Code

A hackathon project I worked on with Ryan Higgins. It is a browser whiteboard app on which you can draw anything. I then worked on a playback option so that it can show exactly what the user draws. Also included are a clear and random-color generator options.

Magic 8 Ball

Java: Android

Download

Packages

Fun project I worked on over because my friend is indecisive. A response is triggered by either a tap on the screen or a shake of the phone or tablet. I looked up how to access the accelerometer in order to get the shaking option to work.
