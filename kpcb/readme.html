Qtqueue

By Andrew Fang

Speed

Qtqueue was designed for fast enqueues and fast dequeues. What's that? Did someone say O(1) for an insertion? Not a problem. O(1) for removal? You got it.

Built with Primitives

Everything was built from the ground up. No lists, no hashsets, no existing data structures. Pure as pure can be, with ints and custom classes.

Bounded

Qtqueue is bounded so you can control the max number of elements that are put in. Want your Qtqueue to only hold 10 elements? Qtqueue(10). Done.

Tested

Qtqueue has been tested throughly by Qttest.py, which extends from Python's unittest. Qttest.py can be viewed and run from with this kpcb folder.

Python

Constructed with Python for readability and flexibility.

Useage

Navigate to the kpcb folder where Qtqueue.py sits, and create any python file. To use the Qtqueue, simply add the following to the top of your .py file:

from Qtqueue import Qtqueue

Constructor

Qtqueue is bounded, so give the constructor a integer to set the max number of elements

myQtqueue = Qtqueue(10)

Enqueue

To add a new item to the Qtqueue, simply call the enqueue method with a number you want to insert

myQtqueue.enqueue(1)

Dequeue

To remove an item, simply call the dequeue method. The oldest item in the queue will be returned.

myQtqueue.dequeue()
