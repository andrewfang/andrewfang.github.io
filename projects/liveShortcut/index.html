Live Shortcut

Hook into Live Activities

Usage

Use the app or create your own Shortcut to remind yourself to do something based on a trigger of your choice.

For example, you can have a Shortcut run when you are away from home, which checks if your garage door is open. Using Shortcut automations, you can send yourself a note that prominently displays on the Lock Screen and the Dynamic Island (if available).
